Manual KTX Seat Helper

KTX 수동 빈자리 확인 도우미

자동 브라우저 조작을 사용하지 않습니다. KORAIL은 평소 쓰는 Chrome에서 직접 로그인/조회하고, Playwright 같은 자동 브라우저 조작을 사용하지 않습니다. KORAIL은 평소 쓰는 브라우저에서 직접 로그인/조회하고, 이 화면은 반복 확인 타이머와 조회 결과 텍스트 판독으로 빈자리를 빨리 찾도록 돕습니다.

KORAIL 직접 열기
중요: 자동 새로고침이 아니라 “수동 반복 확인”입니다.

KORAIL 화면을 프로그램이 자동으로 새로고침하거나 클릭하면 매크로 탐지에 걸릴 수 있습니다. 그래서 이 버전은 Chrome에서 사람이 직접 조회하는 것을 전제로, 정해진 간격마다 “지금 직접 새로고침/조회하세요”라고 알려주고 사용자가 복사해 온 결과에서 빈자리 후보를 찾아줍니다.

Local KTX Booking Helper

KTX 예매 보조 프로그램

성인 1명 기준으로 지정된 KTX 구간을 반복 조회하고, 일반실 또는 특실 좌석이 보이면 예약 버튼을 눌러 결제 직전 단계까지 최대한 이동합니다.

매진 상태에서 반복 확인하는 순서
  1. KORAIL 직접 열기를 눌러 평소 쓰는 Chrome에서 직접 로그인합니다.
  2. KORAIL 직접 열기를 눌러 평소 쓰는 Chrome, Edge, Safari 등에서 직접 로그인합니다.
  3. 아래 조건 카드의 날짜/구간/시간대로 KORAIL에서 직접 열차를 조회합니다.
  4. 수동 새로고침 도우미에서 확인할 조건과 간격을 고르고 반복 확인 시작을 누릅니다.
  5. 타이머가 끝나면 KORAIL 탭에서 사용자가 직접 새로고침 또는 조회 버튼을 누릅니다.
  6. 조회 결과 화면에서 Ctrl+A 후 Ctrl+C로 복사하고, 아래 입력칸에 붙여 넣은 뒤 빈자리 확인을 누릅니다.
  7. 빈자리 후보가 보이면 반복 타이머가 멈추므로 KORAIL 탭으로 돌아가 직접 예약/결제를 진행합니다.
