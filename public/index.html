KTX Chrome 수동 빈자리 확인 도우미

일반 Chrome에서 KORAIL을 직접 로그인·조회·새로고침하고, 이 화면은 타이머와 붙여넣기 판독만 도와줍니다.

중요: 자동화 브라우저를 사용하지 않습니다

Playwright, Puppeteer, Selenium, 자동 클릭, 자동 새로고침 기능은 넣지 않았습니다. KORAIL에서 매크로로 오인될 수 있으므로 일반 Chrome에서 사용자가 직접 조작하는 방식으로 설계했습니다.

가는 편 2026-05-23 광명 → 부산 시간대: 09:00 ~ 13:00 승객: 성인 1명 / 좌석: 일반실·특실 아무 좌석
오는 편 2026-05-25 부산 → 광명 시간대: 09:00 ~ 15:00 승객: 성인 1명 / 좌석: 일반실·특실 아무 좌석

사용 순서

  1. KORAIL 예매 페이지 열기
  2. 새 탭에서 KORAIL에 직접 로그인합니다.
  3. 위 조건으로 직접 조회합니다.
  4. 매진이면 아래 타이머를 켜고, 알림이 울릴 때마다 KORAIL 탭에서 직접 새로고침/조회합니다.
  5. 조회 결과 화면을 Ctrl + A → Ctrl + C로 복사한 뒤 아래에 붙여넣고 검사합니다.
