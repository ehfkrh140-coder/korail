# KTX 예매 보조 프로그램

개인 사용을 위한 로컬 웹 기반 KORAIL 예매 보조 도구입니다. 성인 1명 기준으로 지정된 KTX 구간을 반복 조회하고, 일반실 또는 특실 좌석이 보이면 예약 버튼을 눌러 결제 직전 단계까지 최대한 이동합니다.

## 포함된 고정 작업

- 2026-05-23 광명 → 부산, 09:00 ~ 13:00
- 2026-05-25 부산 → 광명, 09:00 ~ 15:00

조회 간격은 기본 30초입니다.

## 사용 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3001`에 접속합니다.

1. `로그인 브라우저 열기`를 누릅니다.
2. 열린 KORAIL 브라우저에서 직접 로그인합니다.
3. 대시보드에서 원하는 작업의 `감시 시작`을 누릅니다.
4. 좌석이 발견되면 자동화 브라우저가 예약 화면으로 이동합니다. 결제와 본인확인 단계는 직접 진행합니다.

## 주의사항

- KORAIL 화면 구조가 바뀌면 자동화 선택자를 조정해야 할 수 있습니다.
- CAPTCHA, 본인확인, 결제, 보안 확인은 자동화하지 않습니다.
- 좌석이 보이는 순간 다른 사용자가 먼저 가져갈 수 있으므로 예약 성공을 보장하지 않습니다.
- 너무 잦은 조회를 피하기 위해 기본 조회 간격을 30초로 설정했습니다.
